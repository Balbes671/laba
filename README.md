Лабораторная работа представляет собой мини игру огород, в которой можно сажать растения и поливать землю. 
Для роста определенных растений необходим определенный уровень влаги. Для кактуса необходима сухая клетка, для картошки необходима клетка средней влажности, а для болотника - клетка высокой влажности. Через определенное время растения вырастают
